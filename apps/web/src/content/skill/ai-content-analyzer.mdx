## Overview

AI-powered content analysis that helps you create better articles, blog posts, and marketing copy. Get instant feedback on readability, SEO optimization, and engagement potential.

## Analysis Categories

### Readability
- Flesch-Kincaid score
- Sentence complexity
- Passive voice detection
- Vocabulary diversity

### SEO
- Keyword density and placement
- Meta tag suggestions
- Header structure analysis
- Internal linking opportunities

### Engagement
- Emotional tone mapping
- Call-to-action effectiveness
- Headline strength scoring
- Image alt text recommendations

## Usage

```bash
npx @contentai/analyzer --file article.md
```

Or paste your content directly into the web interface for instant analysis.
