## Overview

Create compelling product roadmaps that communicate vision and strategy. This skill helps product managers align teams and stakeholders around clear, achievable goals.

## Features

### Roadmap Creation
- Timeline-based views (quarterly, monthly, weekly)
- Feature groupings and dependencies
- Milestone tracking
- Risk indicators

### Collaboration
- Real-time team editing
- Stakeholder sharing (view-only links)
- Comment and feedback threads
- Approval workflows

### Integration
- Jira, Asana, Linear sync
- GitHub issues linking
- Slack notifications
- Calendar exports

## Roadmap Views

1. **Timeline View** - Gantt-style visualization
2. **Kanban View** - Status-based columns
3. **List View** - Detailed table with filters
4. **Goals View** - Objective-focused planning
