## Overview

React Doctor is a powerful diagnostic tool that helps you maintain healthy React applications. It scans your codebase for common issues across four key areas:

- **Security** - Detects potential vulnerabilities and insecure patterns
- **Performance** - Identifies bottlenecks and optimization opportunities
- **Correctness** - Catches bugs and anti-patterns in React usage
- **Architecture** - Reviews component structure and code organization

## Usage

Simply run the command in your project directory:

```bash
npx @million/react-doctor
```

The tool will analyze your codebase and generate a detailed report with specific recommendations for each issue found.

## Integration

Add to your CI pipeline to catch issues before deployment:

```json
{
  "scripts": {
    "doctor": "react-doctor --ci"
  }
}
```
