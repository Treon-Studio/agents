## Overview

Convert your Figma designs into responsive, maintainable code automatically. This skill bridges the gap between design and development with intelligent component recognition.

## Features

- **Smart Component Detection** - Automatically identifies buttons, inputs, cards, and more
- **Framework Options** - Generate TailwindCSS, plain CSS, or framework-specific code
- **Design Token Extraction** - Captures colors, typography, and spacing as variables
- **Responsive Conversion** - Generates mobile-first responsive designs

## Supported Frameworks

- TailwindCSS
- Plain CSS
- React
- Vue
- Svelte

## Usage

1. Import your Figma file
2. Select the frames to convert
3. Choose your target framework
4. Copy or export the generated code
