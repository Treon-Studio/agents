## Overview

Build robust data pipelines without writing code. Drag-and-drop interface for creating complex ETL workflows with built-in monitoring and error handling.

## Key Features

- **Visual Builder** - Design pipelines with a flowchart-style interface
- **100+ Connectors** - Connect to databases, APIs, files, and cloud services
- **Transformations** - Apply filters, joins, aggregations, and custom logic
- **Scheduling** - Set up cron-based or event-triggered execution
- **Monitoring** - Real-time logs, metrics, and alerting

## Data Sources

- PostgreSQL, MySQL, MongoDB, Snowflake, BigQuery
- REST APIs, GraphQL
- S3, Google Cloud Storage, Azure Blob
- Kafka, RabbitMQ

## Output Destinations

- Data warehouses
- BI tools
- APIs
- File storage
